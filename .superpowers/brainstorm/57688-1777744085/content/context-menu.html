Behaviour notes

  • Header in the menu shows the item name + size + type — so you always know exactly what you right-clicked, even on a small tile
  • Rescan is greyed out with a "dirs only" badge when right-clicking a file
  • Move to Trash triggers a confirmation dialog before invoking the Rust command (same as today)
  • Right-clicking a tile also selects it (dashed outline), so the visual feedback is clear
  • Menu dismisses on outside click or Escape
