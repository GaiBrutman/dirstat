What kind of filtering do you want in the treemap?

You can select multiple if you want more than one

A

Search by name

A text input that highlights or dims tiles whose names don't match. Useful for finding a specific folder/file in a large tree. Already partially implemented in the codebase.

B

Filter by file type / category

Toggle buttons for categories already used in the legend: images, video, audio, code, documents, archives, etc. Click to hide entire category from the treemap.

C

Filter by minimum size

A slider or threshold input (e.g. "hide files smaller than 10 MB"). Focuses the treemap on the big offenders only. Useful when many small files clutter the view.
